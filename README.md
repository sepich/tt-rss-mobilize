tt-rss mobilize
===============

tt-rss plugin to read full feed content mobilized by Instapaper/Readability/Google

### Installation

 1. Copy to:  
/tt-rss/plugins/mobilize/

 2. Activate Mobilize plugin at:  
tt-rss -> Preferences -> Plugins
